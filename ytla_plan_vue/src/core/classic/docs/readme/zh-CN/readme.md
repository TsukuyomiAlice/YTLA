# 经典

### YTLA核心

### Official

version 1.0

前端语言及开发框架: Vue3, typescript  
适用框架: **ytla_plan_vue**   
文件更新日期: 2026-2-12

## 这是一个YTLA的核心

这是YTLA的核心架构的实现核心代码.  
请注意前后端的开发语言和框架.  
你可以获取Official来源的，适用于**classic**核心的应用.

你可以在获取原始版本后对其进行个性化改动.  
但是，不保证个性化改动后的核心与Official来源的，适用于**classic**核心的应用的兼容性

## 概念

本核心为YTLA基于其设计的实现，以实现泛用功能.  
代码遵循了YTLA的框架设计.

### cards

负责卡片功能的后端实现.

### modules

负责模组功能的后端实现.

### plans

负责计划的后端实现.

### busline

作为底层.

### frame

作为页面框架.

### users

负责用户功能.
