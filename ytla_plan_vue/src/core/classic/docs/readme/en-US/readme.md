## This is a YTLA Core

This is the core implementation code of YTLA's core architecture.  
Please note the frontend and backend development languages and frameworks.  
You can obtain applications from Official sources that are compatible with the **classic** core.

You can make personalized changes to it after obtaining the original version.  
However, compatibility between the personalized core and applications from Official sources that are compatible with the **classic** core is not guaranteed.

## Concepts

This core is YTLA's implementation based on its design to achieve universal functionality.  
The code follows YTLA's framework design.

### cards

Responsible for the backend implementation of card functionality.

### modules

Responsible for the backend implementation of module functionality.

### plans

Responsible for the backend implementation of plans.

### busline

Serves as the underlying layer.

### frame

Serves as the webpage framework.

### users

Responsible for user functionality.
